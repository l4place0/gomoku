# Proposal: v2-thread-optimized

## Goal
在已完成5轮训练（第4轮晋升，75%胜率）的基础上，通过优化自博弈并发、训练轮次和评估样本量，加速模型收敛并提升稳定性。

## 背景分析
- 第4轮晋升后第5轮回退至50%胜率
- 根因：价值损失回退（vloss 0.894 -> 1.009）、每轮仅1个epoch、PK样本量不足（20局）
- 自博弈线程数仅4，GPU利用率极低

## 核心改动
1. **自博弈线程数** 4 -> 32，nnMaxBatchSize 64 -> 128
2. **训练epoch** 1 -> 2-3，让价值头充分收敛
3. **PK局数** 20 -> 40-50，降低评估方差
4. **学习率** 0.002 -> 0.001/0.0005，稳定价值损失

## Target Metrics
- Winrate against promoted Round 4 model: >= 55%
- vloss 持续下降，不再回退
- 预计5轮完成，总用时10-15小时

## Evaluation Plan
- 每轮40-50局PK，固定开局，400 visits评估
- 胜率 > 55% 方可晋升
